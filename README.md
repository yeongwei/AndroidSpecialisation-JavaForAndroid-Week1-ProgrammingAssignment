# Overview

Sample Andriod Application that generates various shape (e.g. Triangle, Rectangle, etc) parameters (e.g. Perimeter, Area, etc) based on inputs (e.g Length, Width, Height, etc).

# Notes

1. UI consist of 4 `EditText` for user inputs on shape dimensions (e.g. Length, Width, etc). `Spinner` for drop down shape selection (e.g. Circle, Rectangle, etc). `Button` for processing the inputs. Output to be displayed with `EditText`.

2. `Spinner` dataset setter, see `MainActivity.java`:107.

3. There is only a single Activity which is the `MainActivity`.
